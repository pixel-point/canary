The `CardSelect` component provides an accessible and customizable card-based selection interface. It supports both single and multiple selection modes, making it suitable for forms, settings, and decision workflows.

import { DocsPage } from "@/components/docs-page";

<DocsPage.ComponentExample
  client:only

      </CardSelect.Item>
      <CardSelect.Item value="option-2">
        <CardSelect.Title>Option 2</CardSelect.Title>
      </CardSelect.Item>
    </CardSelect.Root>
  
    <CardSelect.Root type="multiple" defaultValue={["one"]} onValueChange={console.log}>

      <CardSelect.Item value="two">
        <CardSelect.Title>Option B</CardSelect.Title>
      </CardSelect.Item>

  </div>`}
/>

## Usage

```typescript jsx
import { CardSelect } from "@harnessio/ui/components";

// ...

return (
  <CardSelect.Root type="single">
    <CardSelect.Item value="option-1" icon="branch">
      <CardSelect.Title>Option 1</CardSelect.Title>
      <CardSelect.Description>With an icon</CardSelect.Description>
    </CardSelect.Item>

    <CardSelect.Item value="option-2" logo="jira">
      <CardSelect.Title>Option 2</CardSelect.Title>
      <CardSelect.Description>With a logo</CardSelect.Description>
    </CardSelect.Item>
  </CardSelect.Root>
)
```

## API Reference

### Root

The `CardSelect.Root` is the top-level wrapper that controls selection state.

```typescript jsx
<CardSelect.Root
  type="single"                     // Required: 'single' or 'multiple'
  value={selectedValue}             // [OPTIONAL] For controlled usage
  defaultValue={defaultValue}       // [OPTIONAL] For selecting item initially
  onValueChange={handleChange}      // [OPTIONAL] Selection change handler
  orientation="horizontal"           // [OPTIONAL] Card layout direction
  gap="md"                          // [OPTIONAL] Space between cards
  disabled={false}                  // [OPTIONAL] Disable all selections
>
  {/* CardSelect.Item components */}
</CardSelect.Root>
```

### Item

Represents an individual selectable card option.

```typescript jsx
<CardSelect.Item
  value="unique-id"               // Required: Unique identifier
  icon="branch"                  // [OPTIONAL] Icon to display
  logo="gitlab"                  // [OPTIONAL] Logo instead of icon
  disabled={false}               // [OPTIONAL] Disable this item
>
  {/* CardSelect.Title and CardSelect.Description */}
</CardSelect.Item>
```
