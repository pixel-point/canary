The `CopyButton` component provides a button that copies text to the clipboard when clicked. It includes visual feedback to indicate when the text has been copied successfully.

import { DocsPage } from "@/components/docs-page";

<DocsPage.ComponentExample
  client:only
  code={`<CopyButton name="Text to copy" />`}
/>

## Usage

```typescript jsx
import { CopyButton } from '@harnessio/ui/components'

return (
  <CopyButton name="Text to copy" />
)
```
