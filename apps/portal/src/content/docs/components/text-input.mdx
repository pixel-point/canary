The `TextInput` component is used to handle user input, providing options for styling via themes and sizes.
Also offers options for adding icons, captions, custom content, and error text for the field.

## Usage

```typescript jsx
import { TextInput } from '@harnessio/ui/components'

//...

return (
  <TextInput
    size="default"
    value="input with value"
    theme="success"
    caption="Additional information" // displayed as a caption
    error="Error message" // displayed as an error with icon
    inputIconName="plus"
    rightElement={<Button>Action</Button>}
    rightElementVariant="filled"
  />
)
```

## Search Input with State Example

Below is an example of how to implement a search input that displays the current search value using React state. The SearchInput component includes a built-in 300ms debounce to prevent excessive callbacks when typing:

<br />

<SearchInputDoc client:only />

```tsx
import { useState } from "react";
import { TextInput } from "@harnessio/ui/components";

export default function Search() {
  // Using useState hook to track search value
  const [searchValue, setSearchValue] = useState("");

  return (
    <>
      <div className="mb-2 w-full font-bold">Search with State:</div>
      <SearchInput
        placeholder="Type to search..."
        onChange={(value) => setSearchValue(value)}
      />
      <div className="rounded-md border bg-slate-50 p-3">
        <div className="text-sm font-medium">Current Search Value:</div>
        <div className="mt-1 rounded-md border bg-white p-2 font-mono text-sm">
          {searchValue || "(empty)"}
        </div>
      </div>
    </>
  );
}
```
